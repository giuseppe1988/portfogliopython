# Python Portfolio

Portfolio di progetti sviluppati in Python

## Progetti inclusi

### Text to speech Converter - Python
Programma Python che converte un testo scritto in voce e salva l'audio come file.
Il progetto  permette di trasformare input testuali in output audio in modo semplice e automatico

### Expense Tracker - Python
Applicazione Python per la gestione delle spese personali.
Permette di inserire spese,visualizzarle e calcolare il totale in modo semplice e veloce.

### Naval Battle Game
Simulatore di battaglia navale sviluppato in python con gestione turni e logica di combattimento.

### Battle Simulator 
Gioco di combattimento sviluppato in Python con punti vita e attacchi causali.
